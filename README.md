# Proyecto Aplicaciones Ofimáticas - Tienda Online de Moto

## Descripción
Este proyecto incluye la creación de documentos y herramientas ofimáticas necesarias para la gestión de una tienda online de accesorios para motociclistas.

## Documentos incluidos
- Documento descriptivo de la empresa
- Inventario de equipos
- Registro de incidencias
- Presentación del sistema informático

## Objetivo
Organizar la información de la empresa utilizando herramientas como procesadores de texto, hojas de cálculo y presentaciones.
